## 2. 바이브코딩 챌린지 출품 (미니앱 정보 폼)

> 챌린지 주제: **"일상이 편해지는 순간"** = 매일 반복되는 귀찮은 일(예: 더치페이 계산기 두드리기)을 자동화. 우리 앱이 정확히 그 예시를 해결.

### 미니앱 한줄 소개 — 추천
```
영수증만 찍으면 거래가 자동 기록되고, 1인당 얼마인지 자동으로 계산해주는 소모임 간편 가계부예요.
```
**다른 톤 옵션**
```
모임 회비·더치페이 계산, 이제 계산기 없이 자동으로 — 소모임 간편 가계부
```
```
모임 돈 관리(회비·정산·영수증)를 한 곳에서 자동으로 끝내는 소모임 가계부
```

### 챌린지 주제(일상이 편해지는 순간)와의 연관성 — ≈180자 (200자 이내)
```
모임을 하면 매번 영수증을 모아 적고, 계산기로 1/N을 두드리고, 누가 회비를 냈는지 일일이 확인하는 게 번거로워요. 작은 모임은 영수증을 찍으면 거래가 자동으로 기록되고, 1인당 부담액과 정산 대상을 자동 계산해 단톡방에 바로 공유해요. 반복되던 모임 정산·회계의 귀찮음을 없애 일상이 한결 편해지는 순간을 만들어요.
```

**왜 이렇게 썼나**
- 블로그 예시 "더치페이 계산할 때마다 계산기 켜는 것" = 우리 앱이 없애는 불편 → 직접 저격
- 출품 가이드 키워드("반복되는 일 자동화 / 귀찮은 계산·기록 대신")를 그대로 녹임(자동 기록·자동 계산)
- 단순 리워드성 앱이 아니라 핵심 기능(OCR 자동기록·정산 자동계산)이 명확 → 심사 제외 사유 회피
- 1차 심사 = 7/1~26 활성 유저 수(AU). 단톡방 공유·QR/코드 초대가 유입에 유리(앱에 이미 있음)

## 3. (보너스) 콘솔 앱 정보용 초안

### 부제
```
소모임 회비·정산·영수증을 자동으로 정리하는 간편 가계부
```

### 상세 설명 (접속 → 행동 → 결과 흐름)
```
작은 모임은 동호회·여행·정기모임 같은 소규모 모임의 돈 관리를 간편하게 해주는 가계부 앱이에요.

· 토스로 로그인한 뒤 모임을 만들거나, 친구가 보낸 초대 코드·QR로 모임에 참가해요.
· 모임에서 돈을 쓰면 '거래 추가'에서 영수증을 촬영해요 → 거래처·금액·날짜가 자동으로 채워져 기록돼요(영수증 OCR).
· 홈에서 모임의 이번 달 수입·지출·잔액을 한눈에 보고, 거래 탭에서 검색·수정·삭제해요.
· 분석 탭에서 카테고리별 지출·예산·멤버별 1인당 정산 금액을 확인하고, '정산 요청'으로 각자 얼마를 어디로 보내면 되는지 단톡방에 바로 공유해요.
· 회비를 쓰는 모임은 매달 누가 납부했는지 한눈에 관리하고, 연결 계좌도 등록해요.

실제 송금은 하지 않고 '누가 얼마인지'를 자동으로 계산·기록·공유해, 모임 회계의 번거로움을 없애줍니다.
```

### 카테고리(참고)
- 대분류: 생활/편의 · 중분류: 가계부/자산관리 (콘솔 선택지에 맞춰 가장 가까운 것 선택; 다르면 검토 중 안내됨)
